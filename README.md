# fuel
Core Features:
🔐 Login system with military ID (رقم الكنية) authentication
🚗 Vehicle dispatch & return tracking with odometer readings
⛽ Fuel monitoring with visual alerts (green/yellow/orange/red status)
📊 Real-time dashboard showing active trips, overdue vehicles, and maintenance status
📱 Offline capability - queues actions when internet is down, syncs when back online
🔧 Admin panel for updating vehicle counters, status changes (broken/maintenance/delivered), and maintenance logs
📈 History tracking with date filtering
🔑 Password management (self-service and admin override)
